<b>- It allows you to BOLD the required text.

<i>- It allows you to ITALICIZE the required text.

<u>- It allows you to UNDERLINE the required text.

ALIGN- It is used to describe how text is placed on the screen.

<p>-p tag in HTML represents paragraph. Browsers automatically add a single blank line before and after every p element. p tag is an example of container tags. p is used to contain a paragraph of text in a document.

TEXT COLOR-You can change the color of the text inside text color tag.

HEADINGS- There are six levels of headings in HTML. It tells us the hierarchy of the content on a page.

<hr>- The hr tag in html horizontal rule and is used to insert a horizontal line.

bgcolor- The bgcolor attribute is used to set the background color of the webpage.
