Format Style & Heading

HTML provides us with the ability for formatting text just like we do it in MS Word.

<h1> - This is h1 tag which is biggest heading tag in html

<h2> - This is h2 tag second biggest heading tag

<h3> - This is h3 the third biggest heading tag

<h4> - Similarly this is h4 the fourth biggest heading tag

<h5> - Fifth heading tag
<h6> - Smallest heading tag supported in html

<b> - Bold tag: It is used to make certain text or sentences appear bold in webpage. Example: In this sentence this will appear bold

<i> - Italics tag: It is used to make certain word,or sentences italic style. Example: This entire sentence is written in italics style.

<u> - Underline tag: It is used to underline text/sentences. Example: The whole sentence is underlined.

Alignment attribute: use to allign text to right left and centre

<strike> - Strike tag: It is used to strike a word/sentence. Example: This text is striked

<sub> - Subscript tag: Subscript text appears half a character below the normal line. Example: SO2 + O2 + H2O = H2SO4

<sup> - Superscript tag: Subscript text appears half a character below the normal line. Example: (a + b)2 = a2 + 2ab + b2
