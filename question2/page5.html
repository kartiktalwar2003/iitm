Other Tags

<br> - Break tag inserts a single line break.

<hr> - Horizontal tag defines a thematic break in an HTML page (e.g. a shift of topic).

<pre> - Pre tag defines preformatted text.
Example: 
 
                                0    
                          0     0      0
                    0     0     0      0     0

<p> - Para tag defines a paragraph, and is used to write paragraphs in a page with centre, left and right alignment.
