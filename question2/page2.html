Fonts and Tags

Font tag:

The <font> tag plays an important role in the web page to create an attractive and readable web page.
The font tag is used to change the color, size, and style of a text. The base font tag is used to set all the text to the same size, color and face.
 Font tag can make single word to diffrent Style Color and Size.

Basefont tag:

The <basefont> tag is used to change the size of the text.
The basefont tag is used to set all the text to the same size, color and face.
The closing </basefont> tag is not required.
The basefont tag is added to the <head> tag.

Image tag:

The <img> tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages.
The <img> tag creates a holding space for the referenced image.

Example:

src: It is used to specify the path to the image.
alt: It is used to specify an alternate text for the image. It is useful as it informs the user about what the image means and also due to any network issue if the image cannot be displayed then this alternate text will be displayed.
border: It is use to make border around images
height: It is used to specify the height of the image.
width: It is used to specify the width of the image.

If image is not there, wrong path or spelling we can use alt attribute to display an alternative text.
